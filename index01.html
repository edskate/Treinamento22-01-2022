What is the
element in HTML?

The

element in HTML creates a widget that contains additional information that is only visible when the widget is in the “open” state. There are no restrictions on the content that can be included inside the
element.


By default, the widget created by the

element is in the “closed” state. You can click on the widget to toggle between the “open” and “closed” states to reveal or hide the additional information enclosed in the tag.

element, you need to include the element to specify a visible heading for the additional details, as shown below:


01 Attributes

The

element supports the open attribute that specifies whether the content should be visible to the user. The contents are hidden by default. The
element also includes global attributes.


Events

The

element supports all the usual events that are supported by HTML elements. In addition,
also supports the toggle event, which is triggered whenever the
element’s state changes between “open” and “closed”.
